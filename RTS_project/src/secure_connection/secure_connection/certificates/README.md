# certificates/

This directory stores the RSA key pair and per-topic AES key map.

Files generated by `ros2 run secure_connection generate_key`:

| File              | Description                          |
|-------------------|--------------------------------------|
| `private.pem`     | RSA-2048 private key (publisher)     |
| `public.pem`      | RSA-2048 public key (subscribers)    |
| `topic_keys.json` | Per-topic AES-256 key map (hex)      |

⚠️  Keep `private.pem` and `topic_keys.json` confidential.
    Do NOT commit them to a public repository.
